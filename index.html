Bienvenidos a este espacio

Soy la Dra. Catalina Paz Barrientos Cerda, médica con formación en cannabis medicinal. He creado este directorio para facilitar a mis pacientes (y a quien lo necesite) la búsqueda de dispensarios seguros y regulados.

La base de este directorio fue recopilada gracias al trabajo de la Fundación CannabiMed. Mi labor ha sido estructurarlo y ampliarlo en esta web para democratizar el acceso a la información.

Recuerda: El uso de cannabis medicinal debe ser siempre bajo supervisión profesional.

¿Tienes un dispensario o quieres actualizar datos?

Ayudémonos a mantener este directorio como una herramienta confiable para los pacientes. ¡Escríbeme para agregar o corregir información!
